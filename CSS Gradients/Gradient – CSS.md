## Example-1
```
////index.html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Gradient Style</title>
    <!-- <link rel="stylesheet" href="linear-style.css" /> -->
    <link rel="stylesheet" href="linear-style.css"> 
  </head>
  <body>
    <section></section>
  </body>
</html>
////linear-style.css
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

section {
  width: 100%;
  height: 100vh;
  background-image: linear-gradient(red, yellow);
```
